Project Title : Zoofari

##Description
Zoofari is API based platform that gives you all the necessary data of animals.


##Machine Requirement 
npm and node installed, React Development Environment, Internet Connection


##Project Setup
To make the code alive, run command "npm i" in the directory. It will install all the required modules from package.json file.

After the installation, run "npm start"... React will run this project on a port.


Project will get the browser tab and will run automatically.


##Explore the Zoofari
To search about animal click on the Search Tab. It will render a new page.Type the name of an animal in searchbar 
and hit the search button.

A list of animals that matches with your search will be in front of yours.

Click on the animal name to get all the information and pictures of an animal.
